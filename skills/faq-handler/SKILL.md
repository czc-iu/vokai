# FAQ 自动应答

> 智能识别并回答常见问题，提供快速准确的标准化回复

## 触发条件

- 用户询问产品价格、功能、规格
- 用户询问配送、支付、售后政策
- 用户询问公司基本信息、联系方式
- 用户询问账户注册、登录相关问题
- 用户使用"怎么"、"如何"、"什么是"等疑问词

## 使用说明

当识别到常见问题时，按以下优先级处理：

1. **精确匹配** - 如果问题与知识库中的FAQ完全匹配，直接给出标准答案
2. **模糊匹配** - 如果问题与某个FAQ相似度>80%，给出相关答案并询问是否符合需求
3. **关键词匹配** - 根据关键词组合推荐最相关的FAQ列表

### 回答格式

```
【问题类型】xxx

【答案】
xxx

【相关链接】
- [链接描述](URL)

【还有其他问题吗？】
```

### 常见问题分类

- **产品相关**: 功能、规格、价格、对比
- **订单相关**: 下单、修改、取消、状态
- **配送相关**: 时间、范围、费用、追踪
- **支付相关**: 方式、发票、退款
- **售后相关**: 退换货、保修、维修
- **账户相关**: 注册、登录、安全、隐私

## 注意事项

- 保持回答简洁明了，避免冗长
- 如果不确定用户意图，主动询问澄清
- 对于复杂问题，建议转人工客服
- 记录无法回答的问题，用于后续知识库更新
